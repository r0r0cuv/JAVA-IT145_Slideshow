# 🖼️ Java Swing Travel Slide Show (Wellness Edition)

## 📌 Overview
This project is a Java Swing slideshow application built in Eclipse. It displays wellness and detox travel destinations using images and text descriptions.

The application allows users to navigate through slides using Previous and Next buttons.

---

## 🚀 Features
- Interactive slideshow with images
- Text descriptions for each destination
- Previous and Next navigation buttons
- Uses CardLayout for smooth transitions
- Loads images from a resources folder

---

## 🧠 Concepts Demonstrated
- Java Swing GUI development
- Event handling (ActionListener)
- Layout management (BorderLayout, FlowLayout, CardLayout)
- Resource loading (images)
- Application updates based on requirements

---

## ▶️ How to Run
1. Open Eclipse
2. Import the project
3. Make sure images are inside `/images`
4. Run `SlideShow.java`
